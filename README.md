# 🎓 Student Result System 

## 📌 Overview

This is an Excel-based student result system that automatically calculates results based on marks.

## 🚀 Features

* ✔ Pass if all subjects ≥ 33
* ✔ Grace pass if 1 subject < 33
* ✔ Fail if more than 1 subject < 33
* ✔ Fully automated using Excel formulas

## 📂 File Included

* Student-Result-System.xlsx

## ⚙️ How to Use

1. Enter student marks in the sheet
2. Result will be calculated automatically
3. Easy to modify for schools/colleges

## 🛠️ Tools Used

* Microsoft Excel

## 💡 Use Case

* Schools & colleges
* Personal projects
* Excel practice
